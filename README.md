# Pastebin Client - Code and text sharing utility

**Package**: `@apps/pastebin`
**Version**: 0.1.0
**Repository**: `pkg-apps-pastebin`

## Overview

Pastebin Client - Code and text sharing utility

## Installation

### Prerequisites

This package requires the following external commands:

- `curl`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/pastebin

# Or install from local source
cd ~/.pkgs
stow pastebin
```

## Usage

### Application

Pastebin Client - Code and text sharing utility

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/pastebin/` - User configuration
- `~/.local/share/pastebin/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/pastebin

# Or manual unstow
cd ~/.pkgs
stow -D pastebin
```
